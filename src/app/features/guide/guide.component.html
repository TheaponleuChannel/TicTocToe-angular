HOW TO PLAY

NEON GOMOKU — Connect 5

🎯 Objective

Be the first player to place 5 marks in a row — horizontally, vertically, or diagonally — on the grid before your opponent does.

📋 Basic Rules

  • Player X always goes first
  • Players alternate placing their mark on any empty cell
  • First to get exactly 5 in a row wins
  • If the board fills with no winner — the grid expands automatically!
  • You cannot place on an occupied cell

⊞ Dynamic Grid Expansion

The game starts on a 12×12 grid. If all 144 cells are filled with no 5-in-a-row winner, the grid doubles in size (12→24→48→96…). All existing pieces are repositioned to the center of the new, larger grid — play continues!

12×12 → 24×24 → 48×48 → ∞

🤖 VS AI Mode

  • You play as X, the AI plays as O
  • AI uses threat scoring — it tries to win and will block your threats
  • AI prefers center positions and builds multi-directional threats
  • Scores persist across rounds — use the ↺ button to restart with reset scores

🌐 Online Multiplayer

  • Start the backend: node server.js
  • Create Room → get a 6-character code → share it
  • Join Room → enter your friend's code on any device
  • Host is always X, guest is always O
  • Grid expansion is synced in real-time across both devices
  • After each round, both players must request a rematch to reset

🔊 Sound & Controls

  • 🔊 / 🔇 — Toggle sound effects on/off
  • ↺ — Restart the current game (AI mode only)
  • ‹ MENU — Return to main menu (leaves multiplayer room)
  • ? — Open this guide from anywhere in the game

💡 Strategy Tips

  • Control the center — more directions to build sequences from
  • Create dual threats — force opponent to block one while you complete another
  • Watch all diagonals — easy to miss, especially on large grids
  • Block 4-in-a-row immediately — never let opponent get to 4 open ends
  • Think ahead on expansion — plan where pieces will land after grid grows
